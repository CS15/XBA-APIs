Game List

URL Endpoint: /api/games?console={permalink}&letter={alphabeticLetter}&page={pageNumber}
Get Game List by alphabetically
Queries
Console (Required) Console to Search for Games
Letter (Required) Letter to Display Game List
Page Page Number

Example

Endpoint: /api/games?console={{vm.selectedSystem.console}}&letter={{vm.letter}}&page={{vm.pageNumber}}

Total Games: {{vm.games.length}}
